# caMicroscope Code Challenge for Real-time collaborative pathology


## setup

* Clone or download the zip file.
* Run command `yarn` in `chat-frontend` 
* Run command `npm install` in `chat-server` 
* Run command `node index` in `chat-server` 
* Run command `yarn start` in `chat-frontend` 
* Open up `localhost:3000` on two or more separate  browser windows to chat.
